```
Rick and Morty App
Welcome to the Rick and Morty App! This web application allows you to explore and filter characters from the popular TV show Rick and Morty.
```

```
Introduction
The Rick and Morty App is a simple web application built using React and Bootstrap. It fetches character data from the Rick and Morty API and allows users to view a list of characters with various filtering options. Users can explore character details by clicking on individual characters.
```
```
System Requirements
Before running the app, ensure that you have the following installed on your system:

Node.js (at least version 12.0.0)
npm (Node Package Manager)
```

### Packages installation steps

```
npx create-react-app .

npm install bootstrap

npm install @popperjs/core --save

npm install sass

npm install react-paginate --save

npm install react-router-dom

npm start
```
